🚀 Smart Pass

📌 Description

Smart Pass is an IoT-based water level monitoring system using ESP32.
It detects water level and sends data to a dashboard in real time.

⚙️ Features

* Water level detection (Ultrasonic Sensor)
* ESP-NOW communication (Node A → Node B)
* Firebase data storage
* Blynk dashboard
* Automatic alert (Danger level)

🧠 Working

Node A reads water level and sends data using ESP-NOW.
Node B receives data, controls relays, and sends it to Firebase and Blynk.

🔧 Components

* ESP32 (2)
* Ultrasonic Sensor
* Relay Module
